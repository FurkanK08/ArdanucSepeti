# UI Generation Prompt Templates

Gelecekte AI ajanından herhangi bir özel bileşen istenirken kullanılması önerilen şablon dili:

```text
Lütfen {Görev Adı} için bir React Native fonksiyonel bileşeni yaz. 
Bu bileşen 'tokens.json' dosyasındaki renk sistemini kullanmalı (Örn: Butonlar #FF4F00, Arkaplan beyaz) ve 
özellikle {Detaylı Kısıtlamalar} kuralına uymalıdır.
```
