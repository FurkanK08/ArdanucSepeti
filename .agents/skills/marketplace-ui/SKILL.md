# Marketplace UI Construction Skill

Bu klasör, Agent'ın UI (Arayüz) tasarımlarını gerçeğe dönüştürürken kullanacağı katı sistemleri barındırır.
Attığın örnekte olduğu gibi; `examples` klasöründe örnek bileşenler, `resources` klasöründe tasarım tokenları ve `scripts` klasöründe test betikleri bulunur.
Gelecekte arayüz kodlarken herhangi bir belirsizlikte Agent otomatik olarak dönüp bu klasördeki `.json` tokenlarını okumak ve ona uygun sayfa kodlamak zorundadır.
